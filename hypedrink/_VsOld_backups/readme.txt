# 🚀 Hype Drink — Landing Page de Alta Conversão | Herbalife

Landing page otimizada para conversão de clientes do produto **Hype Drink**, operada pelo Distribuidor Independente Herbalife **Edu Sidegum** (Novo Hamburgo/RS). Estruturada com metodologia **AIDA** (Atenção → Interesse → Desejo → Ação), aplicando técnicas de **CRO** (Conversion Rate Optimization), **SEO On-Page** e **acessibilidade (a11y)**.
